#  Person Management System (Full-Stack Docker)
> **SENG 384 - Software Engineering Assignment | Çankaya University**

This project is a containerized Full-Stack Web Application featuring a **React** frontend, a **Node.js (Express)** backend, and a **PostgreSQL** database.

## Tech Stack
- **Frontend:** React 19, Vite, Axios, React Router.
- **Backend:** Node.js, Express.js, pg (node-postgres).
- **Database:** PostgreSQL 15.
- **Infrastructure:** Docker & Docker Compose.

##  Quick Start
Run the entire system with a single command:
```bash
docker compose up --build
